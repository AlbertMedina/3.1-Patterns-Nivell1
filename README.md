# Patterns

## Description - Exercise statement
Three levels of exercises to work design patterns.

### Level 1
Using Singleton pattern to create an Undo class to manage command history from the console input.

### Level 2
Using Abstract Factory pattern to add international contacts (address and phone number) that handle country-specific formats.

### Level 3
Using Command pattern to perform  some actions for different vehicles.

## Technologies used
- Oracle OpenJDK 21.0.8
- Maven 3.9.9
- IntelliJ IDEA Comunity Edition

## Requirements
- Oracle OpenJDK 21.0.8
- Maven 3.9.9
- Java IDE

## Installation
1. Clone repository (https://github.com/AlbertMedina/3.1-Patterns-Nivell3.git).
`git clone https://github.com/AlbertMedina/3.1-Patterns-Nivell3.git`
2. Import project to Java IDE.

## Execution
1. Open project with Java IDE.
2. Locate the three packages (level1, level2, level3).
3. Inside each package, locate and open the file "Main.java".
4. Run each one from the IDE.
5. Tests for each level can also be found in the test folder.
